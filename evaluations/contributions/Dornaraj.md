Describe the features:
* API V2 Frontend

*  User Registration: Users can create an account using POST /api/auth/signup.
*  User Login: Users log in with username and password using POST /api/auth/login. A JWT token is stored in localStorage.
* View VehicleRental: All users can VehicleRental listings using GET /apiVehicleRentaljobs.
*  View VehicleRental Details: Users can see detailed VehicleRental information using GET /api/VehicleRental /:id.
*  Create VehicleRental (Protected): Logged-in users can create VehicleRental using POST /api/VehicleRental.
* Update VehicleRental  (Protected): Authenticated users can edit VehicleRental using PUT /api/jobs/:id.
* Delete VehicleRental  (Protected): Authenticated users can delete VehicleRental using DELETE /api/VehicleRental /:id.
* Authentication: The frontend sends the JWT token in request headers to access protected routes.
